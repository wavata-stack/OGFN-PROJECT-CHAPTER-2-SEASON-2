# Project Setup

Thank you for using this project.

## Installation

Setting everything up is very simple. You only need to connect all components together:

* **Backend**
* **Launcher**
* **Game Servers**

Make sure all of them are configured to communicate with each other correctly before starting the project.

## Game Servers

The game servers are **hardcoded for Chapter 2 Season 2**.

To play on the servers, you **must inject the provided game server DLL/module** before launching the game. Without the injection, you will not be able to connect properly.

The servers are hosted in **EU (Frankfurt)**.

## Backend

The backend includes the following commands:

* `/create`
* `/add`
* `/fulllocker`
* `/ban`
* `/unban`
* `/details`
* `/delete`

Feel free to modify or extend the backend to fit your own needs.

## Launcher

The launcher is a classic game launcher that includes:

* Home
* Library
* Settings
* Logout

You are free to customize the launcher however you like. Change the design, add new features, or improve the user experience to match your own project.

## Customization

Both the **Backend** and the **Launcher** are fully customizable. Edit, improve, or redesign them however you want.

Please note that the **Game Servers are currently hardcoded to Chapter 2 Season 2**, so changing the game version will require additional development.

## Notes

* Connect the Backend, Launcher, and Game Servers correctly.
* Inject the game server module before playing.
* Servers are located in **EU (Frankfurt)**.
* Backend and Launcher are intended to be customized to your preferences.
* Game Servers are fixed to **Chapter 2 Season 2**.

Enjoy the project! 🚀
